Multipage Static Portfolio Website

 Overview

This project is a multi-page static portfolio website, built exclusively using **HTML5** and **CSS3**. The primary goal was to implement best practices in **information architecture**, **visual hierarchy**, and **responsive design** to create a clean and adaptive user experience across various device sizes.

 Website Structure and Pages

The website consists of four required pages, all linked via a unified navigation bar.

| Page | Description |
| :--- | :--- |
| **`index.html`** | The Homepage, featuring a main tagline and a short introduction to the portfolio's owner. |
| **`about.html`** | A detailed personal bio, professional profile image, and a list of key technical skills. |
| **`projects.html`** | A showcase page featuring at least three distinct project cards, each with an image, description, and a 'Details' link. |
| **`contact.html`** | Contact information, including primary details and social media/professional icons. |
